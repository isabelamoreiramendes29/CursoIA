# CursoIA
Este repositório foi criado para documentar minha trajetória de aprendizado no curso de Inteligência Artificial. Nesta primeira etapa, foi realizada uma Análise Exploratória de Dados (EDA) utilizando um conjunto de dados de manutenção preditiva de equipamentos industriais. O objetivo foi conhecer melhor os dados, verificar sua qualidade, identificar padrões e explorar possíveis relações entre as variáveis por meio de estatísticas descritivas e visualizações gráficas.

Ao longo das próximas aulas, novas técnicas e conceitos de Inteligência Artificial serão aplicados sobre este e outros conjuntos de dados, incluindo etapas de pré-processamento, treinamento de modelos de Machine Learning, avaliação de desempenho e desenvolvimento de soluções preditivas. Este repositório será atualizado continuamente para registrar as atividades, análises e conhecimentos adquiridos durante o curso.
